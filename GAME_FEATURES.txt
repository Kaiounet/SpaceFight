CORE GAMEPLAY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [✓] Player Movement
      └─ WASD or Arrow Keys for movement
      └─ Screen boundary collision
      └─ Smooth real-time position sync

  [✓] Weapon System (Beams)
      └─ Press SPACE to shoot
      └─ Mouse aiming (fires toward mouse position)
      └─ 200ms cooldown between shots
      └─ Beams visible as small rectangles
      └─ Beams disappear when hitting players or leaving map

  [✓] Health System
      └─ Each player starts with 100 HP
      └─ Each beam deals 10 damage
      └─ Health displayed above player as color-coded bar
      └─ Green = healthy (>50 HP)
      └─ Yellow = medium (>25 HP)
      └─ Red = critical (≤25 HP)

  [✓] Killing & Respawning
      └─ Players auto-respawn when HP reaches 0
      └─ Respawn at starting position with 100 HP
      └─ Killer receives +10 score points
      └─ Respawn synchronized to all players

  [✓] Scoring System
      └─ +10 points per kill
      └─ Score persists until disconnect
      └─ Score displayed on scoreboard
      └─ Score updates in real-time

  Test 1: Basic Gameplay
    1. Start server and 2 clients
    2. Both players move around
    3. One player shoots at the other
    4. Verify beam appears and travels toward mouse
    5. Verify hit reduces target health by 10
    6. Verify scoreboard shows updated kills

  Test 2: Respawn & Points
    1. Start server and 2 clients
    2. Shooter fires 10 beams at target (100 damage = kill)
    3. Verify target respawns with 100 HP
    4. Verify shooter's score increases by 10
    5. Verify scoreboard is updated on all screens

  Test 3: Multiple Players
    1. Start server and 3+ clients
    2. Multiple players shoot each other
    3. Verify all players see all beams
    4. Verify all players see correct health
    5. Verify scoreboard reflects all kills
    6. Verify no desync between players

  Test 4: Disconnection
    1. Start server and 3 clients
    2. One player kills another (verify score update)
    3. Disconnect one client
    4. Verify disconnected player disappears
    5. Verify remaining game continues normally
